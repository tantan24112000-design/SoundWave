🔥 CONNECT FIREBASE
Để Explore sync real-time giữa tất cả users,
bạn cần kết nối một Firebase Realtime Database miễn phí.
Hướng dẫn (2 phút):
1. Vào console.firebase.google.com
2. Tạo project mới (hoặc dùng project có sẵn)
3. Vào Build → Realtime Database → Create database
4. Chọn Start in test mode (public read/write)
5. Copy URL dạng https://xxx-default-rtdb.firebaseio.com
6. Paste vào ô bên dưới
